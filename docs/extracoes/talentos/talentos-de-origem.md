# Extracao: Talentos de Origem

## 1. Definicao operacional

Nesta extracao, talentos de origem sao os blocos localizados sob o cabecalho Talentos de Origem no HTML local. A propria pagina os descreve como escolhas ligadas a criacao do personagem.
